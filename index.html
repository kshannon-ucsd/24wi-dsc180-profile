About This Demo

This interactive demo showcases our sepsis detection system that combines chest X-ray analysis with patient vitals to predict sepsis risk. Upload an image or use our example data to see the model in action.

Sepsis Detection Model Demo

Our system uses a two-stage approach to predict sepsis risk:

  • First, we analyze chest X-rays using a ResNet model to detect lung anomalies
  • Then, we combine these results with patient vitals for comprehensive risk assessment
